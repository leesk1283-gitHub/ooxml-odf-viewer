# Chrome 확장 프로그램 디버깅 가이드

## 문제: 아이콘을 눌러도 아무것도 실행되지 않음

### 해결 방법 1: 개발자 도구로 오류 확인

1. Chrome에서 확장 프로그램 아이콘을 **우클릭**
2. "팝업 검사" 또는 "Inspect popup" 선택
3. 개발자 도구 창이 열리면 **Console 탭** 확인
4. 빨간색 오류 메시지가 있는지 확인
5. 오류 메시지를 복사해서 알려주세요

### 해결 방법 2: 확장 프로그램 페이지에서 확인

1. `chrome://extensions` 페이지 열기
2. "OOXML/ODF Viewer & Editor" 확장 프로그램 찾기
3. "오류" 버튼이 있는지 확인 (빨간색)
4. 있다면 클릭해서 오류 내용 확인

### 해결 방법 3: 백그라운드 페이지 확인

1. `chrome://extensions` 페이지에서
2. 확장 프로그램의 "서비스 워커" 또는 "백그라운드 페이지" 링크 클릭
3. Console에서 오류 확인

### 일반적인 문제들:

#### 문제 1: 아이콘 파일 경로 오류
- **증상**: "Could not load icon" 오류
- **해결**: 아이콘 파일이 `dist` 폴더에 있는지 확인

#### 문제 2: manifest.json 오류
- **증상**: 확장 프로그램이 로드되지 않음
- **해결**: manifest.json 문법 확인

#### 문제 3: JavaScript 오류
- **증상**: 팝업이 열리지 않거나 빈 화면
- **해결**: Console에서 오류 확인

## 임시 해결책: 새 탭에서 열기

만약 팝업이 작동하지 않는다면, 새 탭에서 여는 방식으로 변경할 수 있습니다.
이 경우 manifest.json을 수정해야 합니다.
